# @ai-paper/admin - 后台管理应用

面向内部运营团队的 PC 管理后台，基于 Vue 3 + Vite + TailwindCSS 构建。

## 功能模块

- **统计看板**：订单趋势、漏斗分析、收入分布（日/周/月切换）
- **订单管理**：查询/筛选失败订单、查看生成任务日志、执行重试/取消
- **模板管理**：学校格式模板的增删启停、处理用户申请
- **用户管理**：用户查询、风控配置（禁用、日生成上限）
- **商品管理**：商品版本管理、上下架控制
- **支付记录**：支付流水查询
- **API 配置**：查重/引用核对服务商配置（密钥始终掩码显示）
- **系统配置**：维护模式、默认建议等
- **后台账号**：账号管理、固定角色权限（超管/运营/客服/只读）
- **操作日志**：所有管理操作的审计日志

## 开发

```bash
# 安装依赖（从 monorepo 根目录执行）
pnpm install

# 启动开发服务器（端口 5174）
pnpm --filter @ai-paper/admin dev

# 运行单元测试
pnpm --filter @ai-paper/admin test

# 构建
pnpm --filter @ai-paper/admin build
```

## 环境变量

通过 vite 代理 `/api` → `http://localhost:3000`，无需额外配置。

生产环境需配置：
- `VITE_API_BASE_URL`：可选，若与代理不同时设置

## 技术规约

- 仅支持 PC 桌面浏览器（宽屏 >= 1280px）
- 使用 session cookie 认证（`admin_session`），不使用 JWT
- 所有密钥在界面中始终以掩码显示（`****xxx`）
- 商品变更不影响历史订单快照
- 生成任务超 2 小时仅告警，不自动终止
