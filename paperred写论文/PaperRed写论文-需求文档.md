# PaperRed 写论文 — 需求文档

> 基于 PaperRed 毕业初稿生成 UI 截图整理的操作步骤与功能需求。

---

## 一、产品概述

- **产品名称**：PaperRed 学术写作模型（融合 DeepSeek R1 等模型）
- **核心功能**：毕业初稿生成（一键快速生成原创文章，重复率 + AIGC 率双达标）
- **流程形态**：左侧步骤导航 + 右侧当前步骤内容，顶部有「标题 → 文献 → 大纲 → 浏览/下载」进度条。
- **服务入口**：顶部导航含 AI写作、降重复率、降Aigc率、免费查重、我的订单、PPT创作、科研工具、AI研读、格式精修等。
- **生成后能力**：写作完成后支持「下载报告」与「在线改稿」；改稿在「PaperRed 智能论文助手」中完成，含内容插入、修文、文本优化、文献管理、智能分析及保存下载。

---

## 二、写论文主流程（五步）

### 步骤 1：录入基础信息

**目标**：收集题目、学历、字数、语言、格式等，为后续生成奠定基础。

| 需求项 | 说明 |
|--------|------|
| 标题/关键词 | 必填。输入完整文章标题；无题目时可点「智能选题」推荐。示例占位：「基于X射线衍射的晶体结构分析技术及其优化探讨」。 |
| AI 投喂 | 选填。建议输入研究思路、关键词、核心观点、研究方法、参考材料等，字数限制 0/1500。 |
| 开题报告上传 | 选填。按钮「↑我有开题/上传开题报告文件」，仅支持 docx/doc；上传后需在输入框核对识别内容与开题一致再确定。 |
| 语言 | 必选。单选：中文 / 英文。 |
| 学历 | 必选。单选：专科 / 本科 / 硕士 / 博士。 |
| 字数 | 必选。如：4000字左右、6000字左右、8000字左右、10000字左右。 |
| 图表公式 | 必选。纯文本 / 图表/公式/代码；提示「在大纲处可手动选择图表/公式/代码」。 |
| 格式模板 | 选填。先选学历，再按学校全称搜索；找不到学校时引导通过右侧「客服与反馈」提交模板或先生成再找客服套格式。 |
| 用户协议 | 必选。勾选「我已阅读并同意：规范使用 AI 工具生成的范文内容，仅用于参考。」 |
| 操作 | 点击「下一步」进入步骤 2。 |

**约束/提示**：标题与 AI 投喂需全部使用英文（若产品要求如此）；上传较慢需提示耐心等待。

---

### 步骤 2：精准知识投喂

**目标**：让模型更准确理解研究需求，提升生成质量。

| 需求项 | 说明 |
|--------|------|
| 投喂内容 | 研究思路、研究方法等；可用豆包、DeepSeek 等生成后再粘贴。 |
| 开题报告 | 上传开题报告后，系统可参考其内容生成大纲与全文。 |
| 与步骤 1 关系 | 步骤 1 已含「AI 投喂」「上传开题」入口，本步可视为强化或补充说明，具体以实际 UI 为准。 |

---

### 步骤 3：确定参考文献

**目标**：选定或导入论文引用的文献，满足学历对应的数量建议。

| 需求项 | 说明 |
|--------|------|
| 文献来源 | ① 系统推荐中英文真实文献；② 用户粘贴知网文献（查新/引文格式）。 |
| 数量建议 | 本科 15 个以上 / 硕士 20 个以上 / 博士 30 个以上。 |
| 自定义输入 | 支持按「标准引文格式」输入，每行一条；一次最多上传 50 份；需提供格式示例与校验提示。 |
| 推荐文献列表 | 每条展示：标题、摘要、作者、期刊/年份、关键词标签；每条约选复选框。 |
| 已选计数 | 底部显示「已选择文献 X 个」。 |
| 扩展搜索 | 提供「没有想要文献? 点此搜索」入口。 |
| 文献确认弹窗 | 标题「文献确认」；提示「请确认您选择或导入的文献是否齐全，如导入的文献缺失请确认导入文献的格式是否正确」；列表展示推荐/已选文献（作者、标题、来源等）；按钮「返回修改」「确认」。 |
| 导航 | 「上一步」回步骤 2，「下一步」进入步骤 4。 |

---

### 步骤 4：确认论文提纲

**目标**：确定论文结构，支持编辑、字数控制及图表/公式/代码占位。

| 需求项 | 说明 |
|--------|------|
| 提纲来源 | ① 免费生成千字提纲（推荐提纲）；② 粘贴已有提纲。 |
| 提纲展示 | 显示当前论文标题；大纲可树形/层级展示（如 一、二、三 + 2.1、2.2…）。 |
| 编辑能力 | 章节标题可编辑；每章节字数可微调，范围 100～2000 字；支持拖拽排序、删除章节。 |
| 插入占位 | 每章节右侧可点亮图标：插入数据(表)、插入图、插入公式、插入代码。 |
| 推荐提纲限制 | 每新建一篇写作，拥有 2 次推荐提纲次数。 |
| 操作按钮 | 「重新生成」「粘贴心仪提纲」；「上一步」「下一步」。 |
| 确认弹窗 | 标题「确认图、表、公式、代码插入信息」；展示已插入数据(表)、图、公式、代码的数量；展示「预估生成字数」（由提纲字数汇总）；提示「AI 将根据提纲生成正文，请完善提纲各级要素，需要图表公式的应在对应章节点亮图标」；按钮「返回修改」「确定」。 |

---

### 步骤 5：确认生成版本与支付

**目标**：选择服务版本、完成支付并进入订单/下载流程。

| 需求项 | 说明 |
|--------|------|
| 版本选择 | 基础版 / AI 率保障版 / 无限改稿版（名称与价格以实际 UI 为准）；可展示价格与功能对比。 |
| 增值与团购 | 邀请同学享团购价；赠品可含：在线改稿、科研绘图、积分、免费查重、AIGC 检测等。 |
| 论文预览页 | 展示标题、目录、部分参考文献；正文为「全文完整内容—付费后可解锁生成、下载」等锁定提示。 |
| 商品信息 | 订单号、商品类型（如毕业论文-本科）、单价、版本说明。 |
| 结算 | 显示总金额、应付金额、R 币余额（若有）；「扫码支付」等支付入口。 |
| 支付成功页 | 成功图标 +「支付成功」文案；按钮「查看订单」「返回」。 |

## 三、生成后下载

**触发条件**：订单状态为「写作完成」时，在「我的订单」列表或订单详情中提供下载入口。

| 需求项 | 说明 |
|--------|------|
| 下载入口（订单列表） | 在「写作完成」的订单行右侧提供绿色按钮「**下载报告**」，点击即可下载当前论文（如 Word 文档）。 |
| 下载入口（改稿页） | 在「PaperRed 智能论文助手」在线改稿界面右侧边栏底部提供红色大按钮「**保存下载**」（带下载图标），用于保存当前编辑内容并下载。 |
| 与改稿关系 | 用户可先「在线改稿」再「保存下载」，或直接从订单列表「下载报告」获取最新已生成版本。 |
| 未完成/未支付 | 订单为「灵感写作(未支付)」或「创作中」时，不展示下载按钮，仅展示「去支付」或等待状态。 |

## 四、订单与订单列表

| 需求项 | 说明 |
|--------|------|
| 我的订单 | 入口在顶部导航；可按服务类型筛选（如 AI写作、查重、降重、PPT、简历、科研绘图、格式精修）。 |
| 订单列表 | 每单展示：订单号、删除按钮、标题（含 Word 图标）、商品类型、状态、提交时间、操作按钮。 |
| 订单状态 | 如「写作完成」「创作中(已支付)」「灵感写作(未支付)」等；未支付显示「去支付」。 |
| 批量操作 | 行选复选框 + 底部「全选」「删除」支持批量删除。 |
| 分页 | 支持分页浏览订单列表。 |

## 五、在线改稿（PaperRed 智能论文助手）

**入口**：在「我的订单」中，对状态为「写作完成」的订单，提供红色提示「如果对当前某部分内容有修改意见,可点击在线改稿前往修改」及红色按钮「**在线改稿**」。点击后进入智能论文助手页面。

**界面结构**：左侧为论文正文编辑区（含摘要、各章节）；右侧为固定侧边栏「PaperRed 智能论文助手」，包含资源状态、文档编辑区、意见修改、文本优化、文献管理、智能分析及底部操作。

### 5.1 资源与入口

| 需求项 | 说明 |
|--------|------|
| 在线改稿次数 | 侧边栏顶部展示「在线改稿次数: X」，表示剩余可用的在线修改/生成次数。 |
| 修改辅导卡券 | 展示「修改辅导卡券数量: X」。 |
| 购买资源包 | 提供红色按钮「购买资源包」，用于购买更多改稿次数或卡券。 |

### 5.2 文档编辑区 — 内容插入

在侧边栏「文档编辑区」下提供「内容插入」：图片、表格、公式、代码。点击后弹出对应弹窗，填写后「提交生成」，生成完成后可插入到正文。

| 功能 | 弹窗/表单 | 说明 |
|------|-----------|------|
| **插图** | 标题（可预填论文标题）、需要结合的内容(0/2000)、提示词(0/1000)、AI出图类型、提交生成 | 生成中显示「正在深度思考中...(已用时Xs)」；生成完成后展示「插图结果」，提供「重新生成」「立即插入」。 |
| **插表** | 标题、需要结合的内容(0/2000)「请输入表格需要包含的数据或文本内容」、提示词(0/1000)「请描述表格的格式、列名、排序或其他特殊要求」、提交生成 | 生成后插入表格到文档。 |
| **插公式** | 标题、需要结合的内容(0/2000)「公式相关的文本内容或数据问题」、提示词(0/1000)「公式类型、格式或其他特殊要求」、提交生成 | 生成后插入公式。 |
| **插代码** | 标题、需要结合的内容(0/2000)「需要通过代码实现的文本内容或问题描述」、提示词(0/1000)「代码风格、注释详细程度等」、编程语言下拉、提交生成 | 生成后插入代码块。 |

### 5.3 意见修改 — 修文

| 需求项 | 说明 |
|--------|------|
| 功能说明 | 「修文」：根据导师建议逐条修改；**不支持修改图、表格、公式、代码**（仅针对正文文本）。 |
| 修文弹窗 | 标题、需要修改的内容(0/2000)、提示词(0/1000)「修文要求如学术风格、专业程度、表达方式等」、修文的预期字数、提交生成。 |

### 5.4 文本优化

侧边栏「文本优化」提供多种 AI 能力，用于对选中或当前段落/章节进行优化：

| 功能 | 说明 |
|------|------|
| 润色 | AI 润色，优化表达与流畅度。 |
| 扩写 | AI 扩写，增加内容长度与细节。 |
| 缩写 | 概括或精简文本。 |
| 降重 | 降低重复率/查重率。 |
| 降AI | 降低 AI 痕迹，适配知网/维普等检测；弹窗可选「降AI类型」：知网、维普、PaperRed、格子达、其他；输入「需要修改的内容」(0/2000)，提交生成。 |
| 翻译 | 文本翻译（标 NEW）。 |

### 5.5 文献管理

| 功能 | 说明 |
|------|------|
| 引用 | 一键插入格式化引用。 |
| 引用文献弹窗 | 标签：「自定义输入参考文献」「搜索文献」。自定义输入：知网查新(引文格式)格式、建议不超过10个文献、解析文献格式按钮；需修改的内容「请输入需要插入引用的文本内容」、引用文献内容修改按钮。 |
| 研究综述 | AI 自动生成相关研究综述。 |

### 5.6 智能分析

| 功能 | 说明 |
|------|------|
| 图片解读 | AI 智能分析图表内容。 |

### 5.7 底部操作

| 需求项 | 说明 |
|--------|------|
| 修改历史 | 按钮「修改历史」，查看历史修改记录或版本。 |
| 保存下载 | 红色大按钮「保存下载」，保存当前编辑并下载论文（与第三节「生成后下载」衔接）。 |

## 六、订单与改稿流程小结

- **下载**：订单「写作完成」→ 订单列表点击「下载报告」，或在改稿页点击「保存下载」。  
- **修改**：订单「写作完成」→ 点击「在线改稿」→ 进入智能论文助手 → 使用内容插入（图/表/公式/代码）、修文、文本优化（润色/扩写/缩写/降重/降AI/翻译）、文献管理（引用/研究综述）、智能分析（图片解读）→ 需要时点击「修改历史」→ 最后「保存下载」。

## 七、非功能与体验需求（从 UI 推断）

| 类型 | 需求要点 |
|------|----------|
| 进度可见 | 左侧固定步骤列表 + 顶部「标题→文献→大纲→浏览/下载」进度条，当前步高亮。 |
| 引导与约束 | 每步有简短说明（如选择学校模板、填写完整标题、推荐文献数量）；必填项用 * 标出；错误或格式问题用红色/醒目提示。 |
| 协议与合规 | 生成内容仅作参考的协议勾选为必选；可配合「规范使用 AI 工具」等说明。 |
| 客服与反馈 | 找不到学校模板时，引导通过右侧悬浮「客服与反馈」联系客服提交模板或后续套格式。 |

## 八、操作步骤汇总（用户视角）

**生成论文**  
1. **录入基础信息**：选格式模板 → 填标题（或智能选题）→ 选填 AI 投喂/开题 → 选语言、学历、字数、图表选项 → 勾选协议 → 下一步。  
2. **知识投喂**：补充研究思路、方法或开题内容（若步骤 1 未填完整）。  
3. **确定参考文献**：选系统推荐文献和/或粘贴/上传自定义文献 → 满足数量建议 → 在弹窗中确认文献 → 下一步。  
4. **确认提纲**：生成或粘贴提纲 → 编辑章节、字数、排序 → 在需要处点亮图表/公式/代码图标 → 在确认弹窗中核对插入信息与预估字数 → 确定。  
5. **选择版本与支付**：在详情页选基础版/AI 率保障版/无限改稿版 → 确认金额 → 扫码支付 → 支付成功后可查看订单或返回。  
6. **订单与成果**：在「我的订单」查看状态，等待「写作完成」。

**下载与修改**  
7. **下载**：订单状态为「写作完成」时，在订单列表点击「下载报告」直接下载；或先进入「在线改稿」编辑后再点击「保存下载」。  
8. **在线改稿**：点击「在线改稿」进入智能论文助手 → 在正文区编辑 + 使用右侧「内容插入」（图/表/公式/代码）、「意见修改」（修文）、「文本优化」（润色/扩写/缩写/降重/降AI/翻译）、「文献管理」（引用/研究综述）、「智能分析」（图片解读）→ 需要时查看「修改历史」→ 最后点击「保存下载」。
