# 产品治理文档 (Product Steering)

## 产品愿景

「傻瓜式 AI 写论文」是一款面向在校学生与科研人员的向导式论文写作辅助 Web 应用。让用户像填表一样，通过六个线性步骤完成从基础信息录入到论文生成、改稿与下载的全流程，无需具备专业写作经验。

## 目标用户

- 主要用户：在校本科生、硕士生、博士生
- 次要用户：科研人员、需要撰写学术论文的职场人士
- 运营人员：负责维护学校格式模板的后台管理员

## 产品形态

- 微信公众号内 H5（移动端优先）
- PC Web（响应式适配，同一套代码）
- 移动端竖屏为主要设计基准，主流程在手机竖屏下均可完成

## 核心流程（六步向导）

1. 步骤 1：录入基础信息（学科、标题、语言、学历/类型、字数、学校/格式）
2. 步骤 2：确定参考文献（系统推荐 + 用户粘贴/上传）
3. 步骤 3：确认提纲（AI 生成 + 用户编辑）
4. 步骤 4：预览与支付（套餐选择 + 微信支付）
5. 步骤 5：后台生成论文（异步任务，按章生成）
6. 步骤 6：改稿和降重（富文本编辑、AI 改稿、查重）

## 业务约束

- 支付方式：仅微信支付（公众号/小程序 JSAPI + PC Native 扫码）
- 退款策略：不支持退款，支付完成即视为服务开始
- 身份认证：微信授权登录（公众号 OAuth），PC 端微信扫码登录，无需单独注册
- 草稿保留：未支付草稿永久保留，用户主动删除；支持多草稿并行
- 后台生成：不设超时限制，持续运行直至全文生成完成
- 引用约束：正文引用仅来自步骤 2 用户确认的文献列表

## 套餐与商业模式

| 套餐 | 说明 |
|------|------|
| 基础版 | AI 改稿操作合计不超过 3 次 |
| AI 率保障版 | 保障论文 AI 检测率低于约定阈值，超出时自动触发降 AI 痕迹处理 |
| 无限改稿版 | AI 改稿操作不限次数 |

## 成功指标

- 用户完成率：从步骤 1 到步骤 6 下载的完成率 ≥ 60%
- 支付转化率：进入步骤 4 的用户支付转化率 ≥ 40%
- 生成成功率：后台生成任务成功率 ≥ 99%
- 引用溯源率：生成正文中可溯源引用比例 ≥ 95%
- 移动端可用性：所有主流程在手机竖屏下可完成，触摸热区 ≥ 44pt

## 非功能性需求

- 性能：AI 生成操作超过 5 秒须显示进度反馈；页面首屏加载 ≤ 3 秒
- 安全：用户数据与微信账号绑定，多端数据互通；支付流程符合微信支付安全规范
- 合规：论文生成内容须符合学术诚信相关法规；查重接入合规第三方服务
- 可用性：后台生成服务 SLA ≥ 99.9%；支持网络中断后重试，不丢失表单数据

## 功能优先级

| 优先级 | 功能 |
|--------|------|
| P0（必须） | 六步向导主流程、微信登录、微信支付、后台生成、Word 下载 |
| P1（重要） | 改稿编辑器、引用核对、查重集成、格式模板 |
| P2（增强） | AI 绘图插图、智能选题、开题报告解析、微信消息通知 |
| P3（后续） | 多语言支持扩展、更多第三方文献 API 接入 |
